# Analise de Tiroteios Policiais nos EUA 2015-2024
Dashboard interativo em Power BI construído a partir da base de dados Washington Post Fatal Force Database, que reúne registros de tiroteios fatais envolvendo policiais nos Estados Unidos entre 2015 e 2024.

**Sobre o projeto**
Este projeto teve como objetivo transformar um conjunto de dados públicos e sensíveis em um painel analítico claro, permitindo explorar o perfil das vítimas, a distribuição geográfica e temporal dos casos, e padrões relevantes ao longo dos anos.

**Objetivos**
Praticar modelagem de dados e DAX em um cenário com dados reais e desafiadores.
Construir um dashboard multi-página com navegação intuitiva.
Extrair insights demográficos e temporais a partir de dados brutos.

**Estrutura do dashboard**
O relatório é dividido em 3 páginas:

Visão Geral — indicadores principais e panorama consolidado do período.
Perfil das Vítimas — recortes demográficos (idade, gênero, raça, estado de saúde mental, entre outros).
Distribuição Geográfica e Temporal — mapas e séries temporais, incluindo a métrica de Média Diária de ocorrências.

**Fonte de dados**
Washington Post Fatal Force Database — base pública mantida pelo Washington Post.
